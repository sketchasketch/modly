# Architecture Decisions

This directory stores approved architecture decision records for Modly.

Conventions:
- File names use a simple numeric prefix: `0001-...md`
- Status values are:
  - `approved`
  - `superseded`
  - `proposed`
- Each record should capture:
  - decision
  - context
  - consequences
